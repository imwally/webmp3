## webmp3

![screenshot](https://raw.githubusercontent.com/imwally/webmp3/master/screenshot.png)

A Go backed HTML5 web-based MP3 player.

## Features 
- [x] Play MP3s (and MP3s only)
- [x] Pause
- [x] Volume increase / decrease
- [x] Show current duration time
- [ ] Skip to position
- [ ] Skip song 
- [ ] Playlist support
